# V1 vs V2 Feature Heatmap Samples

This folder contains 20 test-set visualization samples for each dataset:

- `nuaa/`: NUAA-SIRST
- `nudt/`: NUDT-SIRST
- `irstd/`: IRSTD-1K

Each image compares V1 and V2 on the same test sample. The first row shows input, ground truth, V1 prediction, and V2 prediction. The following rows show stage-2, stage-3, and stage-4 feature heatmaps with `V1 low`, `V1 high_hat`, `V2 low`, and `V2 high_hat`.

`high_hat` is the gated high-branch response used by the decoder, not the raw residual. In the current MIRFD block this corresponds to the output after `residual -> high_raw -> gate -> high_hat`; the heatmap uses channel-wise `abs().mean(dim=0)`.

The full visualization output is generated by `scripts/visualize_v1_v2_features.py`; only 20 samples per dataset are committed here to keep the repository size manageable. For source diagnosis, run the script with `--mode v2-diagnostic` to visualize V2 `low`, `residual`, `high_raw`, `gate`, and `high_hat` separately.
