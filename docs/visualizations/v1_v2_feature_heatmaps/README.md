# V1 vs V2 Feature Heatmap Samples

This folder contains 20 test-set visualization samples for each dataset:

- `nuaa/`: NUAA-SIRST
- `nudt/`: NUDT-SIRST
- `irstd/`: IRSTD-1K

Each image compares V1 and V2 on the same test sample. The first row shows input, ground truth, V1 prediction, and V2 prediction. The following rows show stage-2, stage-3, and stage-4 feature heatmaps with `V1 low`, `V1 high`, `V2 low`, and `V2 high`.

The full visualization output is generated by `scripts/visualize_v1_v2_features.py`; only 20 samples per dataset are committed here to keep the repository size manageable.
